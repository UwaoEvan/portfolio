Gamezone Mobile Application

This was a project I started back in December 2020 when I first learned React Native. I was the sole creator/developer for the project because it was for learning purposes.

Users can use the app inorder for them to rate differentgames they play. Giving them a score of between 1 - 5

Once a user fills in the form, other people could be able to read their reviews and like them.

I worked on this system for 1 day.

Technologies:
  • - JavaScript
  • - Expo Cli
  • - React Native
  • - React Navigation
  • - React Drawer Navigation
  • - React Stack Navigation
  • - Formik
  • - Yup
