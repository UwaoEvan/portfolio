Memories Web Application

This was a project I started back in December 2020 when I first learned ReactJs. I was the sole creator/developer for the project because it was for learning purposes.

The system stored users sample data for their favourite moments and provided an option of editing the created memories, liking the current memories and deleting the memories from the application.

Users had the ability to view all their memories immediately after creating them.

I worked on this system for 1 week before getting freelance jobs.
The backend is created using NodeJs, Express and MongoDB and the API is hosted in Heroku.
The front end is created using ReactJs and is hosted in Netlify

Technologies:
  • - JavaScript
  • - Node REST API
  • - React JS
  • - MongoDB
  • - Heroku
  • - Netlify
  • - ExpressJs
